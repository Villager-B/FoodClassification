# FoodClassification

スーパーの買い物に使用するメモを種類ごとに分類するショートカット

## Requirement
- iOS/iPadOS Device
- Shortcut App
- Pyto(iOS/iPadOS App)
  - Pandasライブラリを使用できるため
- 料理オントロジー構築プロジェクトデータ
  - [データ](http://nlp.indsys.chuo-u.ac.jp/cgi-bin/cooking/wiki.cgi)
